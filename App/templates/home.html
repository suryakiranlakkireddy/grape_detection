Disease Detection in Grape cultivation using strategically placed carners and machine learning algorithms with a focus on powdery mildew and blotches

Project Abstract

Plant diseases significantly affect agricultural productivity and can lead to major economic losses if not detected at an early stage. In grape cultivation, diseases such as powdery mildew and leaf blotches are among the most common factors that reduce crop quality and yield. Traditional disease detection methods rely on manual observation by farmers or experts, which is time-consuming, labor-intensive, and often inaccurate under varying environmental conditions. The base research work proposes an automated disease detection system that utilizes strategically placed cameras and machine learning techniques to identify grape leaf diseases. The existing system employs image preprocessing techniques such as Histogram Oriented Gradient (HOG) and applies multiple machine learning and deep learning algorithms including Convolutional Neural Networks (CNN), Support Vector Machines (SVM), Random Forest (RF), Decision Tree (DT), and Naive Bayes (NB). A stacking algorithm is used to select the most effective model, where CNN demonstrates superior performance in classifying diseased and healthy grape leaves. This work extends the existing system by introducing a hybrid deep learning approach that integrates multiple transfer learning architectures to improve feature extraction and classification accuracy. The extended system utilizes pretrained deep convolutional neural networks including VGG16, VGG19, ResNet, and Inception models as feature extractors. The extracted deep features from these architectures are combined through an ensemble learning strategy using feature-level fusion. The concatenated feature representations are passed through fully connected layers to perform multi-class disease classification. This hybrid ensemble model leverages the strengths of multiple deep architectures, enabling the system to capture diverse image features and improve generalization capability. The extended approach is trained and evaluated on a labeled plant leaf disease dataset using image preprocessing techniques such as resizing and normalization. Experimental results demonstrate that the hybrid ensemble model improves classification performance compared to individual deep learning models and the original basepaper architecture. The proposed extended system provides a more robust and scalable solution for automated grape leaf disease detection and can support smart agriculture applications by enabling early diagnosis and timely intervention.
